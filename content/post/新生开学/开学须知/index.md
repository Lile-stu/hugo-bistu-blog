# 开学须知

## 奖学金及助学贷款

- 奖学金蛮多的：
  - 新生奖学金：用于奖励在统招计划内被正式录取的优秀本科新生。
  - 国家奖学金：用于奖励特别优秀的学生。
  - 国家励志奖学金：用于奖励品学兼优的家庭经济困难学生。
  - 校级奖学金：校长奖学金、学习优秀奖学金、科技创新奖学金、社会贡献奖学金、文体优秀奖学金等，其中最高奖项奖励额度为10000元/人·年。
  - 具体政策查询渠道：北京信息科技大学本科生招生网（政策咨询栏目）、北京信息科技大学招生办微信公众号。

- 助学贷款与绿色通道：
  - 家庭经济困难的新生可通过绿色通道申请国家助学贷款。
  - 绿色通道申请表及注意事项以新生手册为准

- 资助发放账户（北京银行借记卡）：
  - 学生入学后获得的各类奖学金、助学金及其他资助金均通过北京银行借记卡发放。
  - 北京生源：可提前办理北京银行借记卡，办理入学手续后按学校要求在校内信息系统登记银行卡号。
  - 京外生源：入学后两周内办理北京银行借记卡，并按学校要求在校内信息系统登记银行卡号。

- 以上内容为必带物品，其他请参考：新生财力

## 户籍档案管理及要求

- 户口迁移基本规定
  1. 北京籍新生及新疆和田地区定向计划录取的新生户口不迁入学校。其他非京籍新生采取自愿迁入的原则，仅在入学期间办理；学生学籍在校期间不办理户口迁移手续（转学、退学除外）。
  2. 学生集体户口为非农业集体户口。农业户口的新生毕业后迁回原籍的户口性质是否为农业户口由当地派出所依据政策确定。
  3. 自愿迁入者请到本人户口所在地派出所开具《户口迁移证》，迁往地址为：北京市海淀区北京信息科技大学。
     - 《户口迁移证》所列项目须逐项填写齐全（包括文化程度、职业、婚姻状况等）。
     - 身份证号、姓名、性别、出生日期等必须与高考志愿信息完全一致。
     - 出生地和籍贯须写至地级市或县（直辖市须写至区或县）。
     - 落款章和骑缝章（共四个）须加盖齐全。不符合上述要求的，派出所将不予受理。
  4. 报到时将《户口迁移证》交至学校安稳处户籍办公室，由学校统一办理落户手续。
  5. 新生落户审批时间较长。请妥善保管好身份证并留意有效期，以免影响个人相关事宜办理。

- 组织关系转接（党员、团员）
  - 新生党员、团员须办理组织关系转接。
  - 北京生源新生党员：
    - 报到前由原党组织通过“党员E先锋”系统发起党员组织关系转递申请。
    - “组织名称（或组织编码）”请致电录取学院或组织部查询。
    - 报到时将入党材料交至所在学院党委（党总支）办公室，由学院党委（党总支）统一办理组织关系转接手续。
    - 组织关系转接成功后，方可参加所在学院学生党支部的组织生活和相关活动。
  - 无法在系统中转接的党员和外埠生源新生党员：
    - 报到前由原党组织开具《中国共产党党员组织关系介绍信》。
    - 若组织关系转入学院党组织为“党委”，抬头写“北京信息科技大学xxx学院党委”，去处填写“由xxx（原所在单位党组织）去xxx学院党支部”。
    - 若组织关系转入学院党组织为“党总支”，抬头写“北京信息科技大学党委”，去处填写“由xxx（原所在单位党组织）去xxx学院党支部”。
  - 新生团员：
    - 在原学校办理团组织关系转出手续，入校后统一在“北京共青团”线上办理团组织关系转接。
    - 报到时上交团员证；如团员证丢失，携带由原学校开具的团组织关系介绍信，抬头写“中国共产主义青年团北京信息科技大学委员会”，由所在学院团委统一办理转入。
