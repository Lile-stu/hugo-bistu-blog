# 图书馆介绍

2026 编辑 ：沙河校区图书馆现在有配套楼下咖啡馆，而且藏书丰富（甚至有猫物语？你们都私底下荐购了什么.jpg）。并且可以预约讨论间，会议室，座位上有插座。除了不能过夜，是校内仅次于宿舍床上舒服的地方。

## 图书馆整体介绍
### 空间分布
#### 沙河校区
| 楼层 | 功能空间 | 藏书类型 |
| :--- | :--- | :--- |
| **一层** | 校史馆、特色展厅、西海书苑、漂流书屋、文献加工空间等 | 漂流书籍、特色图书 |
| **二层** | 常规阅览区、精荐书苑、主题书架、阶梯阅读空间 | 报刊、外文书、工具书、新书、主题图书、精选图书（精荐书苑） |
| **三层** | 常规阅览区、西宾文库、青衿文库、经典影苑、数智书苑、发呆区、放空区、沉浸阅览区、自主阅读空间、研讨间 | 社会科学图书（A-K）、教师著作（西宾文库）、学位论文（青衿文库）、信息特色图书（数智书苑） |
| **四层** | 常规阅览区、明德书苑、自主阅读空间、数字阅读空间、研讨间 | 自然科学图书（N-Z，不含TP）、红色主题图书（明德书苑） |
| **五层** | 常规阅览区、研讨间 | 自然科学图书（TP类） |
| **六层** | 常规阅览区、观海书斋、研讨间、仰山讲坛 | 传统经典书籍（观海书斋） |
| **B1、B2** | 密集书库 | 密集排架综合类图书、过刊 |

  - 设立地点：学生公寓内
  - 新校区学二公寓北楼一至六层东侧交流厅，共计7个
  - 新校区学三公寓一至六层西侧自助阅读空间，共计12个（房间号分别为1011、1106、2011、2106、3011、3106、4011、4106、5011、5106、6011、6106）

## 文献检索系统

### 文献检索服务介绍
- 服务地点：新校区文理楼A座东南人口大厅、文理楼B座六层WLB-601室（图书驿站）、文理楼C座东南入口大厅各布置1台检索机供读者使用。
- 温馨提示：读者可通过检索机访问图书馆OPAC系统查询本馆馆藏信息，同时可自助办理委托借阅、续借等。使用检索机可访问本馆部分数字资源：中外文数据库、多媒体学习资源、特色资源库资源、试用数据库等。
